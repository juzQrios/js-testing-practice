# JS Testing Practice

Test Driven Development practice using [Jest](https://jestjs.io/en/)

## Installation

```bash
git clone git@github.com:juzQrios/js-testing-practice.git
cd js-testing-practice
git checkout testing-practice
npm install
npm test
```
